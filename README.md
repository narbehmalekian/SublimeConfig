---

To use my color scheme and settings, copy the files to...

> ### Win:
> `%AppData%\Roaming\Sublime Text 3\Packages\User`

> ### Mac:
> `~/Library/Application Support/Sublime Text 3/Packages/User`

> ### Linux:
> `~/.config/sublime-text-3/Packages/User`
